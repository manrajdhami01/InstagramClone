**InstagramClone**

**InstagramClone** is a photo sharing app similar to Instagram but using Parse as its backend.

## User Stories

- [x] User can sign up to create a new account using Parse authentication.
- [x] User can log in and log out of their account.
- [x] The current signed in user is persisted across app restarts.
- [x] User can take a photo, add a caption, and post it to "Instagram".
- [X] User can view the last 20 posts submitted to "Instagram".
- [X] The user can switch between different tabs - viewing all posts (feed view), compose (capture photos form camera) and profile tabs (posts made) using fragments and a Bottom Navigation View. (2 points)
- [X] User can pull to refresh the last 20 posts submitted to "Instagram".
